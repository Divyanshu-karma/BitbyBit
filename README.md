CALTech-chat - Learn together & Grow together!

CALTech-chat is a JavaScript-based collaborative group study web platform inspired by ChatGPT-style interfaces.
It helps students and learners study together through structured group chats, member-wise discussions, and a clean modern UI.

CALTech stands for Collaborative Active Learning — learning together, actively, and effectively.

Features

GPT-Style Interface
Modern and clean chat UI
User and assistant style messages
Auto-scroll to latest messages

Sidebar Navigation

Collapsible and expandable sidebar
Quick access to new chats and navigation
Smooth open/close animations

Theme Modes

Light Mode
Dark Mode
Bolt Mode (visual support coming soon)

Theme preference saved using localStorage

Group Study System

Create study groups with friends

Add members using email addresses

Maximum 5 members per group

Simple and focused group size

Group Management

“Groups” dropdown in top-right corner

Create Group via popup modal

View Your Groups on a separate page (/your_group)

Member-Wise Chat History

Each group member has separate chat history

Member list shown in sidebar

Dropdown view for individual member chats

Technology Used

HTML

CSS

JavaScript (Vanilla JS)

LocalStorage (for mock data and state)

Purpose

CALTech-chat is built to encourage collaborative learning where students:

Share ideas
Help each other
Learn actively in small groups
